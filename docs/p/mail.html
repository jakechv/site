Setting up a NixOS email server

I used Vultr's VPS; install NixOS with it , or using the ISO included with Vultr. (You'll still have to manually partition and install NixOS though).

There are several nice preconfigured mail server options. I chose and run NixOS 20.09, managed with flakes and merged into my system configurations. for the mail server is superb. might also be useful but could be dated.

As part of the instructions, you'll have to set up a reverse DNS record. how you handle that for Vultr.

On Namecheap, the DNS records took several hours to propagate. The LetsEncrypt certificates in this mail server cannot be issued until your DNS records go into effect, so be patient until this happens.
