Zsh

Zsh, z-shell is a more flexible and configurable shell.

It offers an enhanced scripting language as well as greater configurability and pluggability over the standard shell while still implementing both the POSIX standard and conventions. Used with bash, dash and sh (as the POSIX standard is unfortunately far from complete).

Ultimately I don't think ZSH is good as it's this sort of hacked on super-bash; it doesn't *really* forward the goals of a system
