Building A Computer

Working on building a personal computer that leverages simple ideas from the past. Ideally this uses a principle linux system as its base, with sane defaults and a sane desktop environment built on top of a principled core. Eventually we can move towards our own operating system and core to support the environment rather than relying on Linux; a Lisp-like machine with greater flexibility would be beautiful.
